# NeverMiss

NeverMiss is a calm relationship memory app for people who do not want to forget the moments that matter.

It helps you keep track of birthdays, anniversaries, milestones, and personal memories in one simple place. You can save important dates, see what is coming next, set alerts, and write a thoughtful message when it is time to reach out.

## What It Does

NeverMiss keeps important people and moments close without making the experience feel noisy.

- Save birthdays, anniversaries, milestones, and memories
- See upcoming events and alerts in a simple timeline
- Set reminder times so you know exactly when to act
- Generate a wish or write one manually
- Open WhatsApp with your final message when you are ready
- Use it like a lightweight installable PWA on mobile or desktop

## User Flow

The product is designed to feel straightforward from the first visit.

1. Sign in and create your first entry.
2. Add the person, the occasion, the date, and any optional reminder time.
3. Check the dashboard to see upcoming moments.
4. Open Alerts to see what is next and how much time is left.
5. Open Wishes to generate a message or write one yourself.
6. Edit the message until it feels right.
7. Send it through WhatsApp when you are ready.

## Why It Feels Different

NeverMiss is not trying to be a heavy CRM or a cluttered productivity tool.

It is meant to feel personal, quiet, and useful. The focus is not on storing everything. The focus is on helping you remember the right thing at the right time, and making it easier to show care.

## Simple Setup

1. Install dependencies:

```bash
npm install
```

2. Add your environment values in `.env`.

3. Start the app:

```bash
npm run dev
```

4. Open `http://localhost:3000`
